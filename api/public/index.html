💻 Usage Examples

JavaScript

fetch('/api/movies')
  .then(res => res.json())
  .then(data => {
    console.log(data.movies);
  });

Python

import requests

response = requests.get('YOUR_VERCEL_URL/api/movies')
data = response.json()
print(data['movies'])

cURL

curl YOUR_VERCEL_URL/api/movies
